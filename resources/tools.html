Photosensitive Epilepsy Analysis Tool

trace.umd.edu/peat

This is a free, open source tool and quick to install. You can us it review mp4s for dangerous flashing. It can also use it to make a screengrab, so you can check your animations.

aXe

Deque aXe Downloads

I use the browser plug-ins of these, but there are also versions you can integrate into automated testing. As of June 2019, a mobile version was published for Anroid. I played around with the Android one, mostly for color contrast and target size. Similar to WAVE tool.

WebAIM Color Contrast Checker

webaim.org/resources/contrastchecker

There are tons of contrast checkers out there. I like this one because it gives you the contrast ratio and the breakdown pass/fail for small and large type, plus UI components! You can link to results too which is nice.

WAVE

Feed this tool a website url and it will flag issues like missing alt text and missing or incorrect struture. Can use on WAVE website or with plug-ins for free. There are some paid options for testing at bulk. Similar to axe tool.

HTML Validator

Please follow HTML. The great thing about standards is that they save us work. Behaving in expected ways means assistive technologies can work with your work, plus it's got much better chances of existing in the future.

There are tons of ways to check that your HTML is good (it is after all the bedrock of the web!) I use the W3C Validator.

Hemingway Editor

Online tool with feedback on text like Reading level, passive voice, and other facets of readibility.

Screenreaders

Screenreaders are powerful software and like all software, they can take practice to learn all the ins and outs. If you're not someone who uses a screenreaer all the time, be aware of what might be user error and start to learn the syntax of a screenreader.

CADET - Caption And Description Editing Tool

Free tool for creating writing captions and video description tracks. Tool will let you know estimated speech time for text so you can write scripts to fit into a tracks quite periods. Once you write video description, you will want to narrate it or turn it into computer synthesized speech tracks outside of the tool.
