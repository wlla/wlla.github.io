Players

Be sure to share video through an accessible interface. Read more on the Media Players page.

Prerecorded Videos

Prerecorded videos must have captions and video description.

Live Video

Live video are required to have captions. These are provided by CART, Communication Access Realtime Translation. Relevant WCAG Success Criteria: 1.2.4 Captions (Live). If you later published your livestream, remember to add in the requirements for prerecorded video in the section above.

Captions

Captions should be time-stamped, identify speakers, and capture speech and meaningful sounds.

Closed versus Open Captions

Open captions are burned into the video itself, they are part of the visual content of the video. They cannot be turned off. They are most common in an exhibition type situation. Once they become part of the video, their styling doesn't change.

Closed captions means a user can turn them on or off in the interface. These are the most common type of captions. Captioning includes styling information set by the admin and media player. Some media players also let the user control those display settings, which is nice.

WCAG Success Criteria:

  • 1.2.1 Audio-only and Video-only (prerecorded)
  • 1.2.2 Captions (prerecorded)

Captioning Tools

Technology for computer generated transcription is improving every day. It's starting to be incorporated into more products. Remember, it is only ever a start, computer generated captioning always needs to be reviewed and edited for quality and syntax.

CADET - Caption And Description Editing Tool

Free tool for creating writing captions and video description tracks. Tool will let you know estimated speech time for text so you can write scripts to fit into a tracks quite periods. Once you write video description, you will want to narrate it or turn it into computer synthesized speech tracks outside of the tool.

CADET Info page

Youtube

Any public or private video you load into Youtube, you can use Youtube to create captions. You can write them from scratch or edit computer generated captions. Note: computer generated captions always need to be edited and corrected by a person to check for quality and errrors.

YouTube Help Center articles on Translation and Captioning.

Video Description or Audio Description

Video and audio description are related terms. Audio description is an audio track describing the action happening in a video. Video description is a more generic term than audio description since it might take the form of text.

There are two types of audio description, standard and extended. With standard audio description, the description fits into lulls in the sound of the video. A standard audio described video will be the same length as the non-described version.

With extended audio description, a video will actually be paused to allow for a description. Extended audio description is important when a video doesn't naturally accomodate the level of description needed. A big use case would be an educational video covering a complex topic that requires more room than the lulls. A video with extended audio is going to have a longer run-time than the non-described version.

WCAG Success Criteria

  • 1.2.3 Audio Description or Media Alternative (Prerecorded), Level A
  • 1.2.5 Audio Description (Prerecorded), Level AA
  • 1.2.7 Extended Audio Description (Prerecorded), Level AAA

Video Description Tools

YouDescribe

YouDescribe is a community tool for creating audio descriptions for any Youtube video.

YouDescribe has some guidance on creating useful audio descriptions that can help get you started.

When creating original video content, think about how you can communicate the action on the screen into the video script.

CADET

CADET is a tool for creating captions and video description. Read about CADET in the captioning tools section.

Sign Language

Providing a translation in sign language means providing an additional, synced video with a sign language interpreter. The video could be in a different viewport or overlaid on the primary video.

WCAG Success Criteria

1.2.6 Sign Language (Prerecorded), Level AAA
